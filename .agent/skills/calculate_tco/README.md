# 🚗 calculate_tco

**Tính Total Cost of Ownership (TCO) cho xe ô tô** dựa trên profile từ database CarVista.

## Mô tả

Skill này đọc cấu hình thuế/phí từ bảng `tco_rules` trong MySQL, áp dụng công thức
deterministic, và trả về breakdown chi phí theo năm + tổng TCO.

**Đặc điểm:**
- ✅ Hoàn toàn data-driven — không hallucinate rule hay tax rate
- ✅ Deterministic — cùng input → cùng output 100%
- ✅ Hỗ trợ nhiều thị trường (VND, USD, EUR, JPY...)
- ✅ 3 loại công thức: `rate`, `fixed`, `formula` (per_km, straight_line, declining_balance)

## Cách sử dụng

### Kích hoạt tự nhiên
