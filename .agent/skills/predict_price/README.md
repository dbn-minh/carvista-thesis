# 📈 predict_price

**Dự đoán giá tương lai cho xe ô tô** dựa trên lịch sử giá từ database CarVista.

## Mô tả

Skill này phân tích chuỗi thời gian giá từ bảng `variant_price_history`, áp dụng
**linear regression** deterministic để dự báo giá trong tương lai.

**Đặc điểm:**
- ✅ Data-driven hoàn toàn — không hallucinate giá
- ✅ Fail gracefully — thiếu data (<8 points) → trả `null`, không đoán mò
- ✅ Deterministic — cùng DB snapshot + input → cùng output
- ✅ Trả khoảng dao động `[min, max]` dựa trên volatility lịch sử
- ✅ Confidence score 0–1 phản ánh mức độ tin cậy

## Thuật toán

1. **Window:** Lấy tối đa **12 điểm gần nhất** từ `variant_price_history`
2. **Regression:** OLS (Ordinary Least Squares) trên trục thời gian (tháng)
3. **Volatility:** Standard deviation of price returns (period-over-period)
4. **Range:** `predicted_price × (1 ± volatility)`, clamp ≥ 0
5. **Confidence:** `0.6 × point_score + 0.4 × vol_score`, clamp [0,1]

---

## Ngưỡng Confidence Score

| Score | Ý nghĩa |
|---|---|
| ≥ 0.8 | Tin cậy cao — nhiều điểm, thị trường ổn |
| 0.5–0.8 | Trung bình — nên xem kèm predicted_min/max |
| < 0.5 | Thấp — biến động lớn hoặc ít lịch sử |
