## Chức năng

1. **Fetch** thông tin đầy đủ cho từng variant (make/model/trim/specs/price/rating)
2. **Build** bảng so sánh specs cạnh nhau
3. **Generate** pros/cons dựa 100% vào dữ liệu DB — không hallucinate
4. **Recommend** variant tốt nhất theo scoring: rating (40%) + price (35%) + practicality (25%)
